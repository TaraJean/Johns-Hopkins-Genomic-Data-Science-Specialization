# Johns-Hopkins-Genomic-Data-Science-Specialization
All of the code I have saved from the Genomic Data Science Specialization provided by Johns Hopkins University on Coursera; some written / edited by myself, 
others copied from the course practicals

The Specialization consists of 6 courses: Introduction to Genomic Technologies, Python for Genomic Data Science, Algorithms for DNA Sequencing, Command Line Tools for
Genomic Data Science, Bioconductor for Genomic Data Science, and Statistics for Genomic Data Science

## Courses

### Course 1 - Introduction to Genomic Technologies
This course focused on theoretical introductions to molecular biology, sequencing technologies, computer science, and data science so there is no code to include for
this course

### Course 2 - Python For Genomic Data Science
This course introduced Python for those taking the course that come from Life Science backgrounds with no Python programming background, so most code in the course was
very introductory and I have elected not to upload them for simplicity. (I would upload the final assignment but I am having trouble finding the file)

### Course 3 - Algorithms For Genomic Data Science
This course taught multiple algorithms and included lectures and practicals where the algorithms were implemented through Python. The two main focus areas were
sequence alignment and genome assembly, two foundational problems in genomic data science
The files uploaded are all .py files, some are functions practiced from practicals and others are homework assignments produced myself
